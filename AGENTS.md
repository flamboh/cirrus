# AGENTS.MD

The following are guidelines for coding agents working on this project.

## Project Description

Word Cloud. A web app for creating sessions (a la Kahoot!) in which a host can display words submitted by players in a word cloud.

## Specifications

- Use Bun for PM
- Assume there is already a dev server running

### Backend (Convex)

- The backend will use convex for realtime sync and updates

### Frontend (TanStack Start)

- Use shadcn for all primitives, install new primitives through the CLI `bunx --bun shadcn@latest add button`
- Keep it simple, no glowing, no striking colors that were not requested.
